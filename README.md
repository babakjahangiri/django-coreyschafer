# Django-coreyschafer
CoreySchafer tutotial - Full Stack web app with Django



#### Install environment (PIPENV)
```
pipenv shell
```

#### Generate a requirements.txt file from Pipfile.lock

- with pipenv
 ```pipenv requirements.txt```


#### Run the application
```
python manage.py runserver
```
